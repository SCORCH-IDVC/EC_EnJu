# Evaporative cooling and environmental justice

This repo contains the analysis code for analyses examining who relies on evaporative cooling in Tucson, where those households are, and what that means for public health.

## Structure

```
data/           Raw data (not tracked; see below)
results/        Tables (.csv) and figures (.pdf, .png)
scripts/.       Code used to run analyses
```

## Data
The evaporative cooling dataset is not public yet.
